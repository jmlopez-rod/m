# m github branch_prs

Obtain pull request(s) information associated with a branch.

## Usage

```sh
m github branch_prs [-h] [--pretty] [--yaml] [--owner OWNER] --repo REPO branch
```

<BranchPrsSvg />

## Positional arguments

### branch

Name of the branch to query.

## Options

### --owner &lt;OWNER>

- Default: `env.GITHUB_REPOSITORY_OWNER`.

The Github user or organization that owns the repository.

### --repo &lt;REPO>

The name of the repository.

### --pretty

Format the result using colors and indentation.

### --yaml

Use `yaml` format instead of `json`.
